ATM 프로그램

시작화면 : 로그인

   ID(계좌번호) : 1234-1234
   비밀번호 : ****


사용자는 일반사용자와 관리자로 구분


<<관리자화면>>

   1. 계좌생성  2. 계좌삭제  3. 계좌조회  4. 로그아웃


1. 계좌생성 : 사용자의 정보(계좌번호(중복금지), 이름, 비밀번호, 잔고)를 입력받아 계좌를 생성. -> 메뉴로이동


   => 상속을 이용한 여러 종류의 계좌생성이 가능하다면 가산점있음.

      ex 마이너스 한도 통장, 체크카드 통장, (추가된 통장만큼 프로그램에 기능도 추가되어야할 것)

2. 계좌삭제 : 계좌번호를 입력받아 삭제. -> 메뉴로이동

3. 계좌조회 : 이름으로 오름차순 정렬된 계좌정보(예금주, 계좌번호,  통장종류(여러통장이 있다면))를 1행으로 출력 -> 메뉴로이동


4. 로그아웃 : 시작화면으로 돌아감.


<<일반사용자화면>>

   1.입금  2.출금  3.송금  4.조회  5.로그아웃  6.기타(체크카드기능이 있다면..)


1. 입금 : 입금액을 입력 후 계좌정보를 출력 -> 메뉴로이동


2. 출금 : 비밀번호와 출금액을 입력후 계좌정보를 출력 -> 메뉴로이동


3. 송금 : 상대의 계좌번호와 출금액, 본인의 비밀번호를 입력후 본인의 계좌정보를 출력 -> 메뉴로이동


4. 조회 : 비밀번호 입력후 계좌정보를 출력 -> 메뉴로이동


5. 로그아웃 : 시작화면으로 돌아감.



예러는 최대한 발생하지 않도록 처리할 것.


반드시 이용되어야할 것들.

 참조자, 연결리스트(계좌관리), 파일입출력(데이터보존), string클래스(문자열저장)


그 외에 c++의 기능(업/다운 캐스팅 ,상속, 가상함수, 오버로딩 등)들을 잘활용하면 가산점 있음.


 Account 클래스는 본인 프로그램에 맞도록 조금씩 수정하여 사용해도 됨.



학생들끼리 표절 또는 인터넷 표절시 0점

2인 1조를 기준으로 채점함 (3인 1조인 경우 프로그램도 3인분량이 되어야함)
